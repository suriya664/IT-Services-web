API Integration Guide

Complete guide to integrating with CloudGuard Technologies services

Getting Started

Welcome to the CloudGuard API documentation. Our API allows you to programmatically access your security data, manage services, and integrate with your existing systems.

Base URL

https://api.cloudguard.in/v1

Prerequisites

  • Active CloudGuard account
  • API credentials (available in your dashboard)
  • HTTPS-enabled application

Authentication

CloudGuard API uses API keys for authentication. Include your API key in the request header:

curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.cloudguard.in/v1/security/alerts

Obtaining API Keys

  1. Log in to your dashboard
  2. Navigate to Settings â†’ API Keys
  3. Click "Generate New Key"
  4. Store the key securely (it won't be shown again)
Security Note

Never expose your API keys in client-side code. Use environment variables and keep keys secure.

API Endpoints

Security Alerts

Retrieve security alerts for your organization.

GET /security/alerts 
// JavaScript Example
const response = await fetch('https://api.cloudguard.in/v1/security/alerts', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const alerts = await response.json();
console.log(alerts);

Webhooks

Configure webhooks to receive real-time notifications about security events.

Webhook Events

  • alert.created - New security alert
  • alert.resolved - Alert resolved
  • ticket.updated - Support ticket status changed
  • service.status - Service status change

Webhook Payload Example

{
  "event": "alert.created",
  "timestamp": "2025-10-14T10:30:00Z",
  "data": {
    "alert_id": "ALT-12345",
    "severity": "high",
    "description": "Suspicious login attempt detected",
    "affected_assets": ["server-01", "server-02"]
  }
}

Webhook Security

Verify webhook signatures using HMAC SHA256:

# Python Example
import hmac
import hashlib

def verify_webhook(payload, signature, secret):
    computed = hmac.new(
        secret.encode('utf-8'),
        payload.encode('utf-8'),
        hashlib.sha256
    ).hexdigest()
    return hmac.compare_digest(computed, signature)

Error Handling

The API uses standard HTTP status codes:

Code Meaning Description
200 OK Request successful
400 Bad Request Invalid request parameters
401 Unauthorized Invalid or missing API key
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Server error occurred

Error response format:

{
  "error": {
    "code": "invalid_request",
    "message": "Missing required parameter: priority",
    "details": {
      "field": "priority",
      "reason": "required"
    }
  }
}

Code Examples

Node.js

const axios = require('axios');

const client = axios.create({
  baseURL: 'https://api.cloudguard.in/v1',
  headers: {
    'Authorization': `Bearer ${process.env.CLOUDGUARD_API_KEY}`
  }
});

// Get security alerts
async function getAlerts() {
  try {
    const { data } = await client.get('/security/alerts');
    return data;
  } catch (error) {
    console.error('API Error:', error.response.data);
  }
}

Python

import requests
import os

API_KEY = os.environ.get('CLOUDGUARD_API_KEY')
BASE_URL = 'https://api.cloudguard.in/v1'

headers = {
    'Authorization': f'Bearer {API_KEY}',
    'Content-Type': 'application/json'
}

def get_alerts():
    response = requests.get(
        f'{BASE_URL}/security/alerts',
        headers=headers
    )
    return response.json()
